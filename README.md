# Sales Analytics Pipeline using Azure Data Factory and Power BI

## Project Overview

This project demonstrates an end-to-end sales analytics solution using Azure Data Factory, Azure Blob Storage, and Power BI.

The project automates the ingestion and movement of sales, customer, and product data files into a processed data layer for reporting and analytics.

## Business Problem

Organizations often receive sales data from multiple sources. Manual data movement can be time-consuming and error-prone.

This project automates the process and prepares data for business reporting.
